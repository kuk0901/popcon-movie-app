## 디렉터리 구조

- db/: 데이터베이스 관련 파일(erd)
- flowchart/: 주요 기능의 플로우차트 관련 파일
- pdf/: 기획안을 pdf로 변환한 파일
- ui/: ui 디자인 관련 파일
